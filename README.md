# Crystalix Teleport

___

## Features

- `/tpa <Spieler>` Verschickt eine Teleportanfrage an einen Spieler, welche man über Buttons im
  Chat an- bzw. ablehnen kann. Nach 60 Sekunden läuft eine Anfrage automatisch ab.
- `/tpignore` Toggelt einen Status, welcher verhindert, dass einem Teleportanfragen geschickt
  werden können.
- Multilanguage Support (deutsch und englisch); wählt automatisch basierend auf der eingestellten
  Sprache im Client.
- Persistenz der Ignorierungen. Diese werden über den Serverneustart hinweg in einer `config.json`
  gespeichert.
